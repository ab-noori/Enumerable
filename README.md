In order to run this project you need:

Example command:

```sh
 Install Ruby
```

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone https://github.com/ab-noori/Marse-Code-Decoder.git
```

Install this project with:

```sh
  cd my-folder
  git clone https://github.com/ab-noori/Marse-Code-Decoder.git
```
